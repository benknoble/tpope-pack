# tpope-pack

This repository maintains a collection of vim plugins created by @tpope.
The goal is to make them installable all at once using `vim8` `+packages`.

## Installation

```bash
mkdir -p ~/.vim/pack
cd !!:2
git clone --recursive https://github.com/lgalke/tpope-pack.git
```
